Grid System

檔案名稱：sass/_bootstrap_grid.scss

HyUI 格線系統預設為bootstrap的grid 系統，gutter為15px，一樣以12欄作為格線，如需更改預設格線配置可前往：sass/mixins/_bootstrap_grid.scss調整設定

常見佈局範例

header_div
left_div
center_div
footer_div

<div class="container">
    <div class="row">
        <div class="header_div">header_div</div>
        <div class="left_div">left_div</div>
        <div class="center_div">center_div</div>
        <div class="footer_div">footer_div</div>
    </div>
</div>

在任何SCSS檔案裡面，只需要對應好class輕鬆設定以下就可以製作類似bootstrap的格線系統

                    
.header_div{
  @include make-xs-column(12);  //576px  width:100%
  @include make-sm-column(12);  //768px  width:100%
  @include make-md-column(12);  //992px  width:100%
  @include make-lg-column(12);  //1400px width:100%
  @include gutter();
}
.left_div{
  @include make-xs-column(12);  //576px  width:100%
  @include make-sm-column(6);   //768px  width:50%
  @include make-md-column(4);   //992px  width:33.3%
  @include make-lg-column(4);   //1400px width:33.3%
  @include gutter();
}
.center_div{
  @include make-xs-column(12);  //576px  width:100%
  @include make-sm-column(6);   //768px  width:50%
  @include make-md-column(8);   //992px  width:66.6%
  @include make-lg-column(8);   //1400px width:66.6%
  @include gutter();
}
.footer_div{
  @include make-xs-column(12);  //576px  width:100%
  @include make-sm-column(12);  //768px  width:100%
  @include make-md-column(12);  //992px  width:100%
  @include make-lg-column(12);  //1400px width:100%
  @include gutter();
}
